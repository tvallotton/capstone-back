# Setup
1. [instala docker desktop](https://www.docker.com/products/docker-desktop/)
2. ejecuta `docker compose build`
3. ejecuta `docker compose up -d`

# Correr el back
Para correr el back ejecuta: 
```sh
docker compose up -d
```
